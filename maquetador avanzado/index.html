Nuevos Elementos HTML5

Incorporaciones al estandar

Con el surgimiento de html5 se agregaron nuevas caracteristicas e implementaron nuevas etiquetas, entre los elementos que se incorporaron se encuentran herramientas estadisticas, la posibilidad de reproducir contenido multimedia de forma nativa y ademas se agrego la capacidad de poder realizar distintos tipos de graficos dentro del documento (vectoriales y mapa de bits). Si bien en la actualidad la mayoria de los navegadores soporta estos elementos, podemos corrobar si los mismos son compatible desde el sitio Can I Use

Estadisticas

Por otro lado <canvas> (Lienzo) permite la creacion de graficos raster (mapa de bits) o imagenes pixelares dentro de nuestro documento utilizando como lenguaje JavaScript permitiendonos realizar interacciones con el usuario, gracias a las caracteristicas del mismo.

Preprocesadores

Pseudolenguajes de CSS

Son herramientas que permiten generar hojas de estilos a partir de una sintasis particular, utilizan algun lenguaje de programacion como herramienta de compilacion para realizar dicha tarea. Estos programas poseen las mismas caracteristicas que el lenguaje CSS, pero agregan alguna funcionalidades que el mismo no posee, ademas que su sintaxis suele ser mas corta en comparacion de la hoja de estilos resultante.

LESS (Leaner Style Sheets)

Preprocesador que utiliza Javascript como lenguaje de compilacion, ademas brinda a las hojas de estilos algunas caracteristicas del mismo como la asincronicidad e invocacion de metodos. Para instalarlo como aplicacion, debemos ingresar en la consola el comando de nodejs npm i -g less y para compilar las hojas de estilos deberemos ingresar el comando lessc "input.less" "output.css"

El unico inconveniente que se nos presenta a la hora de trabajar con LESS es que este no posee un metodo de observacion, es decir, que debemos compilar nuestros archivos cada vez que realicemos cambios. Una solucion a esto es utilizar un plugin conocido como Less-Watch-Compiler que lo instalamos utilizando el comando npm i -g less-watch-compiler y para compilar nuestros archivos ingresamos el comando less-watch-compiler "input/" "output/ "file.less"

SASS (Syntactically Awesome Styles Sheets)

Preprocesador que utiliza Ruby como lenguaje de compilacion, a diferencia de LESS, no posee la capacidad de invocar funciones o metodos si estos no fueron previamente declarados. Varia en la forma que se generan los mixins y distingue con sus sintaxis entre los diferentes tipos de elementos.

Para instalarlo debemos ingresar el comando de NodeJS npm i -g sass y para compilar nuestros archivos debemos ingresar el siguiente comando sass --watch "input.scss" "output.css" el cual ofrece un metodo de observacion para una compilacion automatica. Existen 2 formatos de archivo, siendo el primero sass, que no utiliza la mayoria de los signos y se maneja con la tabulacion; y luego esta scss cuya sintaxis es similar a css o less.
